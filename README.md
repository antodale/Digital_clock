# Digital Clock
An arduino based digital clock with fun lamp options activate through the switches. 
Necessary components are: 
 - Arduino nano board,
 - RTC external component for Serial communication,
 - a configurable number of programmable multicolor leds,
 - 2 toggle swithces,
 - 5 10k resistors,
 - 1 photoresistor.
